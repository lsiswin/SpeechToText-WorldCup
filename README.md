# 🎙️ 智能多功能媒体工具箱 & 🏆 2026世界杯模拟预测系统

这是一个基于 **Vue 3 + Vite** 前端与 **ASP.NET Core (C#) Web API** 后端构建的模块化多功能媒体工具箱系统。界面采用深邃优雅的暗黑科技风，配备高保真玻璃微光质感与微交互动效。

本系统集成了语音转写、在线媒体下载、视频压制翻译、以及 2026 美加墨世界杯的高级模拟推演模块。

---

## ✨ 核心功能模块

### 🎙️ 1. 本地音视频转文字 (Whisper 离线版)
* **全离线语音识别**：内置 `Whisper.net` 库，首次运行自动下载 Base 识别模型，数据全部在本地处理，安全可靠。
* **高兼容格式转换**：自动检测/下载本地 FFmpeg，将任意音视频格式转为标准 16kHz PCM 单声道 WAV 格式进行高精识别。
* **富文本交互校对**：段落按精确时间轴展示，支持点选播放、波形寻道、双击修改文本内容。
* **多格式一键导出**：支持导出美观的 Word (.docx)、SRT 字幕及普通 TXT 文本。

### 📥 2. 在线视频提取与转录 (YouTube & Bilibili)
* **智能链接解析**：输入 YouTube 或 Bilibili 链接，自动拉取高清视频封面、标题、播放时长及作者头像信息。
* **分流高速下载**：支持单独提取音频轨或下载 1080P/720P 高清 MP4 视频，下载完毕自动清理临时文件。
* **视频语音转录**：下载后直接调入 Whisper 本地引擎，实现一键由网络视频生成精细文本。

### 🎬 3. 视频智能字幕压制与翻译 (DeepSeek / Google)
* **智能双语翻译**：可选免密钥 Google 翻译或接入 **DeepSeek API** 进行高精准度语境翻译。
* **ASS 高保真字幕生成**：自动计算时间轴，生成明黄色字体加黑色背景板的高清硬字幕。
* **实时控制台**：配备磨砂玻璃质感的实时终端日志播放器，压制过程与进度一目了然。

### 🏆 4. 2026 美加墨世界杯晋级预测系统
* **完整官方对阵落位**：内置 48 支国家队及官方 A-L 小组初始分配，Round of 32 淘汰赛严格遵循官方树状匹配规则。
* **官方真实赛程时间**：内置揭幕战及 72 场小组赛真实北京时间表。
* **蒙特卡洛（Monte Carlo）出线概率推演**：根据每支球队的实力评分（Strength: 60-95），输入比分后自动在后台运行 500 次算法模拟，实时反馈晋级概率、已晋级或已淘汰状态。
* **DeepSeek AI 真实模拟 & 中文战报弹窗**：支持“全局一键模拟”或“小组单体模拟”，AI 会分析球队形势并生成极其精美的中文局势总结，用户点击“确认落实”后才会应用比分。
* **海报级高画质分享图片导出**：采用 Canvas 2D 绘图，通过 CORS 异步缓存加载矢量国旗，一键导出深蓝线性渐变与大力神杯金光的 800x1020 预测海报。

## 🛠️ 技术栈

* **前端**：`Vue 3` + `Vite` + `Vanilla CSS` + `HTML5 Canvas` (CORS 异步处理)
* **后端**：`ASP.NET Core 8.0` + `Whisper.net` + `YoutubeExplode (v6.6.0)` + `Open-XML (Word)` + `FFmpeg` 自动部署器

## 🚀 快速启动指南

为了方便您的使用，项目根目录下已为您编写好了一键批处理启动脚本：

### 1. 服务一键启动
* 双击运行根目录下的 **`run-all.bat`**：一键安装所需依赖，并同时唤起前端与后端服务。
* *或手动启动*：
  * 后端启动：双击运行 `run-backend.bat`（或在 `backend/SpeechToText.Api/` 执行 `dotnet run`）
  * 前端启动：双击运行 `run-frontend.bat`（或在 `frontend/` 执行 `npm run dev`）

打开浏览器访问：`http://localhost:5173` 即可进入主页。

### 2. 配置 DeepSeek API 密钥（可选）
本系统对密钥文件进行了安全保护，绝不会提交到公共仓库。若您想使用 DeepSeek AI 翻译及世界杯推演总结功能：
1. 在 `backend/SpeechToText.Api/` 文件夹下新建文件 **`appsettings.local.json`**。
2. 写入以下内容：
   ```json
   {
     "Translation": {
       "Provider": "DeepSeek",
       "ApiKey": "您的真实DeepSeekAPI密钥"
     }
   }
   ```
3. 保存后重启后端服务即可。
